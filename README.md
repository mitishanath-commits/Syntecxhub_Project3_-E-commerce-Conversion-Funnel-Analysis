*Funnel Analysis Project*
An interactive funnel analysis project using Python and Jupyter Notebook to analyze customer journey stages from Browse → Purchase.
*Features* 
-> Funnel stage analysis
-> User drop-off detection
-> Conversion rate calculation
-> Bottleneck identification
-> Interactive visualizations using Matplotlib, Seaborn & Plotly

*Insights*
-> User count decreases at each funnel stage
-> Major drop-offs identified before purchase
-> Helps improve conversion optimization strategies

*Technologies Used*
-> Python
-> Pandas
-> Matplotlib
-> Seaborn
-> Plotly
-> Jupyter Notebook

*Project Workflow*
-> Data Cleaning
-> Funnel Stage Definition
-> Conversion Analysis
-> Bottleneck Detection
-> Data Visualization
-> Business Insights

*Outcome*
This project helps understand customer behavior, identify weak funnel stages, and improve overall conversion performance.
